Methodology Note

All three models were trained on the Telco Customer Churn dataset with an 80/20 train-test split. Categorical features were one-hot encoded; numerical features standardised via StandardScaler. Random Forest achieved the highest test accuracy at 84%, outperforming XGBoost (83%) and Decision Tree (78%). Random Forest's ensemble of decorrelated trees provides strong generalisation and resistance to overfitting on this dataset.
